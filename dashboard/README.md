# لوحة التحكم 🎛️

لوحة تحكم شاملة لإدارة جميع خدمات متجر أبو سرور كارد

## المميزات الرئيسية

### 1. إدارة الخدمات
- ✅ إضافة وتعديل وحذف الخدمات
- ✅ إدارة الأسعار والعروض
- ✅ تفعيل وتعطيل الخدمات

### 2. إدارة المستخدمين
- ✅ قائمة المستخدمين
- ✅ التحكم في الصلاحيات
- ✅ تتبع النشاط

### 3. التقارير والإحصائيات
- ✅ إجمالي المبيعات
- ✅ عدد العمليات
- ✅ رسوم بيانية توضيحية
- ✅ تقارير يومية وشهرية

### 4. إدارة المدفوعات
- ✅ سجل المدفوعات
- ✅ المبالغ المستردة
- ✅ التسويات المالية

### 5. الإعدادات
- ✅ إعدادات النظام
- ✅ إعدادات الأمان
- ✅ إدارة المفاتيح

## هيكل لوحة التحكم

```
dashboard/
├── admin.js              # صفحة المسؤول
├── services.js           # إدارة الخدمات
├── users.js              # إدارة المستخدمين
├── reports.js            # التقارير والإحصائيات
├── payments.js           # إدارة المدفوعات
└── settings.js           # الإعدادات
```

## الوصول إلى لوحة التحكم

```
URL: /dashboard
URL: /admin
```

## متطلبات الوصول

- اسم المستخدم
- كلمة المرور
- مستوى صلاحيات: Admin
